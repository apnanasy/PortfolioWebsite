This is a simple expense app that allows you to put in your income and expenses in as few clicks as possible. It uses the Room library for database access. It is a stable release on the play store, but is still in active development, with a widget feature soon to be added.

This is a little timer that I built to be used in a call center environment. It allows you to choose between an extended hold at 6 minutes and a regular hold at 3 minutes. Once the timer reaches 30 seconds it warns the user to check back in with their customer by flashing the buttons and timer yellow. once it has gone past the allotted time it will start flashing red. You can view the source code here and click on the timer.js and timer.html files
